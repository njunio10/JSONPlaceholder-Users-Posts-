# Exercises — JSONPlaceholder (Users & Posts)

> Keep it simple and progressive. Do each task in order.  

---

## A) Users — https://jsonplaceholder.typicode.com/users
1) Fetch users using the *users* endpoint and console them in the screen;
2) Fetch users using the *users* endpoint and console table them in the screen;
3) In React, Render users (id, name and username).
4) Improve the design of the page by changing CSS;
5) In React, Render users (name, email, phone and website).
6) Improve the CSS design.
---
## B) Posts — https://jsonplaceholder.typicode.com/posts
1) Fetch users using the *posts* endpoint and console them in the screen;
2) Fetch users using the *posts* endpoint and console table them in the screen;
3) In React, Render users (userId, id and title).
4) Improve the design of the page by changing CSS;
5) In React, Render users (userId, id, title and body);
6) Improve the CSS design.
